# womanrise-academy

## Project Description
WomanRise E-Learning Academy is a web-based e-learning platform developed by a team of women software developers. The system provides flexible and accessible online education for all learners. It allows users to register, browse and enroll in courses, access learning materials, take quizzes, and track their learning progress. Instructors can manage course content, while administrators oversee system operations. The project demonstrates the practical application of software engineering tools, UML modeling, collaborative development, and version control using GitHub.

## Project Objectives
- To design and develop a functional e-learning platform
- To apply software engineering tools and practices learned in class
- To model system requirements using UML diagrams
- To practice collaborative development using Git and GitHub
- To implement and test system components following standard workflows

## Scope of the System
- User registration and authentication
- Course browsing and enrollment
- Learning content access
- Quiz and assessment management
- User, course, and system administration

## Technologies & Tools
- Frontend: React.js
- Backend: Node.js / Java
- Database: MySQL / MongoDB
- UML Modeling Tool: Enterprise Architect
- Version Control: Git & GitHub
- Testing: JUnit
